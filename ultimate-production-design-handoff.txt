6. BACKEND-VERBINDUNGEN
─────────────────────────────────────────────────────────────────────

  PocketBase (über ZimaOS App Store):
    - Admin unter http://<ZIMAOS-IP>:8090/_/
    - Collection: leads (für Kontaktformular + Konfigurator)
    - Collection: contact_messages (für Kontaktformular)

  API-Routen:
    POST /api/contact
      → Zod-Validierung (name, email, phone, subject, message, consent)
      → Rate-Limiting (10/min pro IP)
      → Erzeugt Eintrag in PocketBase contact_messages
      → Error Handling (400, 429, 500)

    GET /api/health
      → Gibt Status, Timestamp, Service-Name zurück

  ENV-Variablen (.env.example):
    NEXT_PUBLIC_POCKETBASE_URL — öffentliche PocketBase-URL
    POCKETBASE_URL — interne PocketBase-URL
    POCKETBASE_ADMIN_EMAIL — Admin-Login
    POCKETBASE_ADMIN_PASSWORD — Admin-Passwort
    NEXTAUTH_SECRET — Session-Secret
    NEXTAUTH_URL — Öffentliche URL
    ADMIN_USER/ADMIN_PASSWORD — Basic Auth für /admin
